pointers for div

  1. Width of block level element is 100% of the viewport width
  2. height of a block isn't set by default
  3. height and width can be specified for block level elements using CSS

pointers for span

  1. Width of an inline level element is dependent on the content
  2. height of an inline isn't set by default
  3. height and width cannot be specified for the inline level elements using CSS
